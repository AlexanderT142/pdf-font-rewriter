# Built-In Fonts

These fonts are bundled into the Obsidian plugin so users can rewrite PDFs without finding a font file first.

License files and upstream notices are kept in `licenses/`. Most bundled fonts use the SIL Open Font License 1.1; XCharter and TeX Gyre Pagella use their upstream TeX package font licenses.

| Font | Style | Source | Local license notice |
| --- | --- | --- | --- |
| Charis SIL | Serif | https://software.sil.org/charis/ | `licenses/charissil-OFL.txt` |
| XCharter | Serif | https://ctan.org/pkg/xcharter | `licenses/xcharter-README-LICENSE.txt` |
| TeX Gyre Pagella | Serif | https://ctan.org/pkg/tex-gyre-pagella | `licenses/texgyrepagella-GUST-FONT-LICENSE.txt` |
| EB Garamond | Serif | https://fonts.google.com/specimen/EB+Garamond | `licenses/ebgaramond-OFL.txt` |
| Inter | Sans serif | https://fonts.google.com/specimen/Inter | `licenses/inter-OFL.txt` |
| Noto Sans | Sans serif | https://fonts.google.com/noto/specimen/Noto+Sans | `licenses/notosans-OFL.txt` |
| Open Sans | Sans serif | https://fonts.google.com/specimen/Open+Sans | `licenses/opensans-OFL.txt` |
| Lato | Sans serif | https://fonts.google.com/specimen/Lato | `licenses/lato-OFL.txt` |
| Atkinson Hyperlegible | Sans serif | https://www.brailleinstitute.org/freefont/ | `licenses/atkinsonhyperlegible-OFL.txt` |
| Andika | Sans serif | https://software.sil.org/andika/ | `licenses/andika-OFL.txt` |
| OpenDyslexic | Sans serif | https://opendyslexic.org/ | `licenses/opendyslexic-OFL.txt` |

Palatino is not bundled because it is proprietary. These bundled fonts are open-license alternatives.
