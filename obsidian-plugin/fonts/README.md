# Built-In Fonts

These fonts are bundled into the Obsidian plugin so users can rewrite PDFs without finding a font file first.

All bundled fonts below are distributed under the SIL Open Font License 1.1. The corresponding license texts are in `licenses/`.

| Font | Style | Source |
| --- | --- | --- |
| Libertinus Serif | Serif | https://github.com/google/fonts/tree/main/ofl/libertinusserif |
| Source Serif 4 | Serif | https://github.com/google/fonts/tree/main/ofl/sourceserif4 |
| Libre Baskerville | Serif | https://github.com/google/fonts/tree/main/ofl/librebaskerville |
| PT Serif | Serif | https://github.com/google/fonts/tree/main/ofl/ptserif |
| Libertinus Sans | Sans serif | https://github.com/google/fonts/tree/main/ofl/libertinussans |
| Atkinson Hyperlegible | Sans serif | https://github.com/google/fonts/tree/main/ofl/atkinsonhyperlegible |
| Fira Sans | Sans serif | https://github.com/google/fonts/tree/main/ofl/firasans |
| Work Sans | Sans serif | https://github.com/google/fonts/tree/main/ofl/worksans |

Palatino is not bundled because it is proprietary. These bundled fonts are open-license alternatives.
